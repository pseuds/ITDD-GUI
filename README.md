# ITDD-GUI

## Overview
GUI for Interlocking Tiles Defect Detection. Made with PyQt5.
- Displays image segmentation prediction from a pre-trained model.
    - Detects the following classes:
        - Interlocking tiles
        - Asphalt
        - Manhole
        - Road paint (fade)
        - Road paint (good)
        - Repair tiles
    - Trained with YOLOv8, 1498 images, imgsz=800
